## Featured Work

### GatesAI Chat
A local-first AI workspace built with React, TypeScript, Tauri, and a sandboxed local bridge. It supports bring-your-own-model workflows, local tools, file operations, web-lite mode, and a polished desktop-style chat interface.

**Why it matters:** shows AI product engineering, local runtime design, tool orchestration, UI polish, and test discipline.  
**Links:** [live web demo](https://calculator5329.github.io/GatesAI-Chat/) · [release repo](https://github.com/Calculator5329/GatesAI-Chat-releases)

### N2K Platform
A full rebuild of an N2K mental-math platform with a solver engine, game modes, CLI tools, competition workflows, binary data formats, and a React/MobX web app.

**Why it matters:** shows algorithmic thinking, architecture, performance work, domain modeling, and product expansion from solver to platform.  
**Links:** [source](https://github.com/Calculator5329/N2K)

### Agent V2
A provider-agnostic agent backend for long-running LLM tool loops. Supports OpenAI, Anthropic, and Gemini-style providers, schema-validated tools, SSE streaming, process management, and extensible tool registration.

**Why it matters:** shows practical AI infrastructure beyond wrapper apps.  
**Links:** [source](https://github.com/Calculator5329/agent-v2)

### Tax Explorer
An interactive dashboard for exploring US tax burden by income group, state, and tax category, with custom charting and historical effective-rate views.

**Why it matters:** shows data visualization, public-policy modeling, and frontend clarity around complex data.  
**Links:** [source](https://github.com/Calculator5329/tax-explorer) · [live](https://tax-explorer-app.web.app)

### Circuit Builder
A visual digital logic circuit designer and simulator built with React, TypeScript, React Flow, and local persistence.

**Why it matters:** shows graph-based UI, visual editing, state management, and simulation-oriented interfaces.  
**Links:** [source](https://github.com/Calculator5329/circuit-builder) · [live](https://circuit-builder-d5ea8.web.app)
